# Professional Personal Portfolio

A clean, modern, static personal portfolio website built using only **HTML5** and **CSS3**.
Features a neon gradient dark theme, responsive design, and PWA manifest.

## Project Structure

```
portfolio-website/
│
├── index.html          # Home Screen
├── about.html          # About & Education
├── skills.html         # Technical Skills Showcase
├── projects.html       # Projects Grid
├── contact.html        # Contact Form & Info
├── offline.html        # PWA Offline Fallback
│
├── css/
│   └── style.css       # Main Stylesheet (Variables, Flex/Grid, Responsive)
│
├── assets/
│   ├── images/         # Placeholders for project images
│   └── icons/          # Icons for Manifest
│
└── manifest.json       # Web App Manifest
```

## Features

- **Zero JavaScript**: Pure styled interactions.
- **Neon Theme**: Modern dark mode with glowing accents.
- **Responsive**: Mobile-first design using Media Queries.
- **PWA Ready**: Includes `manifest.json`.

## How to Run

1. Open `index.html` in any modern web browser.
2. No server required (though static serving is recommended for PWA features to trigger).
